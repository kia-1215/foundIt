# FoundIt 🔍

FoundIt is a full-stack campus lost-and-found platform that allows students to report lost items, browse reported items, and manage their own reports securely.

## Features

- User registration and login
- JWT authentication
- Password hashing with bcrypt
- Report lost items
- Browse all reported items
- Search items
- View your own reports
- Mark items as found
- Delete your own reports

## Tech Stack

- React
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- bcrypt

## Future Improvements

- Image uploads
- Claim request system
- Email notifications
- Admin dashboard

## Installation

```bash
npm install
```

Client:

```bash
cd client
npm run dev
```

Server:

```bash
cd server
node index.js
```
